画像向き自動補正ツール
========================

縦画面で撮影された動画のスナップショット群を、上下を自動判別して
正しい向きに回転させるツールです。

■ 使い方
--------
1. ImageOrientationTool.exe をダブルクリックして起動してください
   （インストール不要です）
2. 初回起動時にWindowsのSmartScreenが警告を表示する場合があります。
   「詳細情報」→「実行」を選択してください。
   （個人開発のためコード署名をしていないための警告で、
   ウイルスではありません）
3. 「📁 画像を読み込む」から画像ファイルを選択してください

4. 自動判定された向きがおかしい場合は、画像を選択して
   手動回転ボタン（左90°／右90°／180°）で補正できます
5. 「💾 元ファイルに反映する」を押すまで、元の画像ファイルは
   一切変更されません。安心してお試しください

■ 主な機能
--------
- EXIF情報・肌色検出・画像特徴分析を組み合わせた自動向き判定
- 顔検出フィルター（顔が写っていない画像を自動で選択）
- ダーク／ライトテーマ自動切替
- 日本語・English・한국어・Deutsch・Русский・Français・简体中文・
  繁體中文(台灣) に対応（メニューバー「設定」→「言語」から切替可能）

■ 動作環境
--------
- Windows 10 / 11 (64bit)
- インストール不要・単体で動作します
